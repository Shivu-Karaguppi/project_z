# Skill: Store Tabular Data

## Description
This skill allows you to provide tabular data that will be stored in your computer's memory files for future reference.

## Usage
- Provide tabular data in any message.
- The skill will parse and save the data in a structured format in memory files.

## Requirements
- No special enabling required. This skill only requires write access to your workspace memory directory.

## Commands
- **store table**: Store the provided tabular data.
- **retrieve table**: Retrieve stored tabular data.

## Notes
- Tabular data should be provided in a reasonable format (e.g., markdown tables, CSV format, or similar).
- Long-term memory updates are done by editing the relevant memory files.
